# Ultimate Stimulation Clicker (100 Upgrades)

**Ultimate Stimulation Clicker** เป็นเกมแนว Clicker/Idle ที่ผสมผสานระบบเศรษฐกิจจำลอง การลงทุน และการประมูลวัตถุโบราณเข้าด้วยกันอย่างลงตัว โดยเน้นประสบการณ์การเล่นที่ "เพลิดเพลิน" (Stimulating) ผ่านการตอบสนองที่รวดเร็วและระบบการอัปเกรดที่หลากหลายกว่า 100 รายการ

---

## 🎮 Concept เกม

"จากมือใหม่หัดคลิก สู่มหาเศรษฐีเจ้าของอาณาจักรหุ้นและวัตถุโบราณ"
ตัวเกมเน้นการสะสมทรัพยากรที่เรียกว่า **Stime Coin ($)** และ **Adee Coin (🪙)** เพื่อนำไปอัปเกรดความสามารถในการหาเงิน ทั้งแบบ Active (คลิกเอง) และ Passive (CPS - Clicks Per Second) พร้อมทั้งท้าทายผู้เล่นด้วยตลาดการลงทุนที่มีความผันผวนตามเหตุการณ์ข่าวสารในเกม

---

## ✨ ฟีเจอร์เด่น (Detailed Features)

### 1. ระบบ UI แบบ Resizable Split-Screen
- ผู้เล่นสามารถลากเส้นแบ่ง (Divider) ระหว่างส่วนการเล่นหลัก (Gameplay) และส่วนร้านค้า (Shop) ได้ตามต้องการ
- รองรับการใช้งานทั้งบน Desktop และ Mobile (Responsive Design) อย่างสมบูรณ์แบบ

### 2. ระบบการเล่นหลัก (Core Gameplay)
- **Tap to Earn:** คลิกเพื่อรับเงินสะสม
- **CPS & CPC:** ระบบคำนวณเงินต่อวินาที (Clicks Per Second) และพลังการคลิกต่อครั้ง (Clicks Per Click)
- **Leveling System:** ระบบเลเวลที่มาพร้อมกับ "ยศ" (Titles) ตั้งแต่ *มือใหม่* ไปจนถึง *พระเจ้าแห่งการกระตุ้น* (Level 999)
- **XP Bar:** ทุกการคลิกและการซื้อไอเทมจะมอบ XP เพื่อเพิ่มเลเวล และรับรางวัลเป็น Adee Coin

### 3. ตลาดการลงทุน (Market Economy)
- **Stocks & Crypto:** ลงทุนในหุ้นเทคโนโลยีระดับโลก (เช่น Anthropic, Alphabet, SpaceX) และคริปโตเคอเรนซี (Bitcoin)
- **Index Funds:** ลงทุนในกองทุนดัชนีที่มีความเสี่ยงต่ำกว่า
- **Live Charts:** กราฟแสดงราคาแบบ Real-time พร้อมประวัติราคาย้อนหลัง
- **Volatility:** ระบบความผันผวนของราคาที่เป็นเอกลักษณ์ในแต่ละสินทรัพย์

### 4. ระบบข่าวสาร (News Ticker)
- ข่าวสารที่ปรากฏในแถบด้านบนจะส่งผลกระทบต่อราคาตลาดโดยตรง (เช่น ข่าวบวกทำให้หุ้นพุ่ง, ข่าวลบทำให้ตลาดหมี)
- ผู้เล่นต้องคอยติดตามข่าวเพื่อหาจังหวะ Buy Low, Sell High

### 5. ตลาดวัตถุโบราณ (Antique Market)
- **Procedural Generation:** วัตถุโบราณถูกสร้างขึ้นแบบสุ่มด้วยระบบอัลกอริทึม ทำให้แต่ละชิ้นมีชื่อ ประวัติ และความหายากที่ไม่ซ้ำกัน
- **Rarity & Grade:** มีความหายากตั้งแต่ Common ถึง Mythic และเกรดตั้งแต่ ของปลอม (Fake) ไปจนถึงของแท้ดั้งเดิม (Ancient True)
- **Appraisal System:** ผู้เล่นสามารถอัปเกรดทักษะ *นักโบราณคดี (Archaeologist)* และ *นักประเมิน (Appraiser)* เพื่อดูข้อมูลเชิงลึกของไอเทมก่อนตัดสินใจซื้อ

### 6. โรงประมูล AI (AI Auction House)
- **AI Bidders:** ระบบประมูลที่จำลองผู้เข้าร่วมประมูลที่เป็น NPC (AI) ที่มีบุคลิกการสู้ราคาที่แตกต่างกัน
- **Sell/Buy Modes:** ผู้เล่นสามารถนำวัตถุโบราณในคลังมาเปิดประมูลขายให้กับ NPC หรือจะเข้าร่วมประมูลไอเทมหายากจาก NPC ก็ได้
- **Real-time Bidding Log:** บันทึกการต่อสู้ราคาที่ดุเดือดในโรงประมูล

### 7. ระบบโฆษณา Adee Network
- ดูโฆษณาเพื่อรับ Adee Coin ซึ่งเป็น Premium Currency สำหรับซื้อไอเทมพิเศษที่หาไม่ได้จากเงินปกติ

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla CSS - ไม่ใช้ Framework เพื่อความเร็วสูงสุด), JavaScript (ES6+)
- **Architecture:** 
    - **DOM Manipulation:** ใช้ Vanilla JS ในการจัดการ UI และ State ของเกม
    - **Asynchronous Data:** ใช้ `fetch` API ในการโหลดข้อมูล JSON สำหรับไอเทมและวัตถุโบราณ
    - **Engine:** ระบบ Game Loop ที่ทำงานแยกส่วนกัน (Score Tick, Market Tick, Auction Tick)
- **Utility Scripts:**
    - **Python:** ใช้สำหรับงาน Data Processing เช่น การดึงประโยค (extract_sentences.py) และการแปลข้อมูลวัตถุโบราณ (translate_artifacts.py)
- **Data Format:** JSON (สำหรับจัดเก็บข้อมูลมหาศาลของไอเทม, หุ้น, ข่าว, และ NPC)

---

## 📊 โครงสร้างข้อมูล (Data Structure)

- `data/items.json`: รายการอัปเกรดกว่า 1,000 บรรทัด ครอบคลุมทั้งไอเทมทั่วไปและไอเทมพิเศษ
- `data/ancientArtifacts.json`: ฐานข้อมูลวัตถุโบราณขนาดใหญ่ (1MB+) สำหรับระบบสุ่ม
- `data/stocks.json`: ข้อมูลพื้นฐานและสถานะของสินทรัพย์ทางการเงิน
- `data/news.json`: คลังเหตุการณ์ข่าวสารที่ส่งผลกระทบต่อเศรษฐกิจในเกม
- `data/npcs.json`: โปรไฟล์และพฤติกรรมของ AI ในโรงประมูล
